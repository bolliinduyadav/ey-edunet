  • Element selectors
  • Id Selectors

The css element selector

All Html tags based on the element name is called as element selector

        Examples :p,h1,h3.img,b,i,etc....
    h1{
        text-align:center;
        color:red;
    }

The css ID selector

The id selector uses the id attribute of an html element to select a specific element.to select an element with a specified id,write the hash(#)character followed by the id of a element

        Examples :HTML element with id="para1"
        #para1{
        text-align:center;
        color:red;
    }

The css class elector

The class selector selects HTML elements with a specific class attribute to select elements with a specified class,write a period(.),followed by the class name, 

        Examples :HTML element withclass="center"
        .center{
        text-align:center;
        color:red;
    }

The css universal selector

The universal selector(*)selects all html elements on the page

        Examples :
        *{
        text-align:center;
        color:red;
    }

The css grouping selector

To group selectors,separate each selectors with comma.

        Examples :
    h1,h2,p{
        text-align:center;
        color:red;
    }
